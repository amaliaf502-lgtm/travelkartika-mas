Interactive Shell:

================================================================================
📱 HALAMAN-HALAMAN WEBSITE
================================================================================

Beranda (Home):
  URL: http://localhost:8000/
  Status: PUBLIC (tidak perlu login)
  Konten: Showcase paket terbaru, info, keunggulan layanan

Daftar Paket:
  URL: http://localhost:8000/pakets
  Status: PUBLIC (tidak perlu login)
  Konten: List semua paket umroh dengan paginasi

Detail Paket:
  URL: http://localhost:8000/pakets/{id}
  Status: PUBLIC (tidak perlu login)
  Konten: Info lengkap, fasilitas, itinerari, harga

Form Pemesanan:
  URL: http://localhost:8000/pemesanan/{paket}/create
  Status: PROTECTED (harus login)
  Konten: Form pesan paket + kalkulasi harga otomatis

Pemesanan Saya:
  URL: http://localhost:8000/pemesanan-saya
  Status: PROTECTED (harus login)
  Konten: List semua pemesanan customer

Detail Pemesanan:
  URL: http://localhost:8000/pemesanan/{id}
  Status: PROTECTED (harus login)
  Konten: Detail pemesanan + opsi batalkan

Login:
  URL: http://localhost:8000/login
  Status: PUBLIC
  Default: test@example.com / password

Register:
  URL: http://localhost:8000/register
  Status: PUBLIC
  Konten: Form daftar user baru

================================================================================
🆘 TROUBLESHOOTING
================================================================================

ERROR: Database connection failed
SOLUSI: Pastikan MySQL/MariaDB running di Laragon

ERROR: Halaman blank atau error 500
SOLUSI: 
  - Jalankan: php artisan config:clear
  - Jalankan: php artisan cache:clear

ERROR: Styling tidak muncul
SOLUSI: 
  - Hard refresh browser (Ctrl+Shift+R)
  - Atau gunakan incognito mode

ERROR: Migration failed
SOLUSI: Jalankan: php artisan migrate:fresh --seed
